# نظام الترشيح الذكي والتفاعلي | Interactive Data Filtering System 🚀 تطبيق تفاعلي مبني بلغة بايثون واستخدام مكتبة Tkinter، يهدف إلى تسهيل عملية تصفية وترشيح البيانات (أرقام ونصوص) بناءً على شروط يحددها المستخدم عبر واجهة رسومية (GUI) بسيطة وذكية. An interactive Python application built with Tkinter for data filtering. It allows users to filter lists of numbers or strings based on custom dynamic criteria through a clean GUI. ## 🌟 الميزات | Features - **ترشيح الأرقام:** فرز الأعداد الزوجية، الفردية، أو الأكبر من قيمة معينة. - **ترشيح النصوص:** تصفية الكلمات التي تبدأ بحرف معين، أو حسب طول الكلمة. - **واجهة رسومية (GUI):** تصميم متجاوب وسهل الاستخدام مع معالجة ذكية للأخطاء (Error Handling). - **سهل التشغيل:** لا يتطلب أي مكتبات خارجية معقدة. ## 🛠️ التقنيات المستخدمة | Tech Stack - **Python 3.x** - **Tkinter** (مكتبة الواجهات الرسومية القياسية) ## 🚀 طريقة التشغيل | How to Run 1. تأكد من تثبيت بايثون على جهازك. 2. قم بتحميل الملف أو نسخ الكود إلى ملف باسم `filter_app.py`. 3. افتح الطرفية (Terminal) وشغل الأمر التالي: ```bash python filter_app.py
